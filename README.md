## 勤怠管理システムのきっかけ

派遣時代に自分で一ヶ月の残業と勤務時間を手書きで提出していて、手間だと感じたから。  
デジタル化できたら便利。あとPCが苦手な人でも簡単だと思って、アナログよりもこちらを使用してくれるようになれば嬉しいから。

## 感想

やり始め、GitHubのURL結合などで時間がかかってしまう。英語表記なので難しいが、調べてみるとアップロードなどもできた。

また、コードを書こうにも何から手を付ければいいかわからないので、AIとの会話で頭を整理しながら、工程を分けてまず入力だけ完成させた。
